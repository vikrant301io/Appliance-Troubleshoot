# Nameplate Example Images

This directory should contain 4 example nameplate images that will be displayed with the nameplate location guidance.

## Image Files Expected:

1. `nameplate_example_1.jpg` - GE Refrigerator nameplate (GENERAL ELECTRIC, Model GXSF30V 00, Serial ZA160697B)
2. `nameplate_example_2.jpg` - GE Range/Cooktop nameplate (MODEL NO: ZEK7500SH2SS, SERIAL NO: BK123456G)
3. `nameplate_example_3.jpg` - GE Range nameplate (MODEL NO. CHP9536SJ3SS, SERIAL NO. ML041307Q)
4. `nameplate_example_4.jpg` - Refrigerator interior showing nameplate location

## Usage:

These images are automatically displayed when the nameplate location guidance is shown to users.
